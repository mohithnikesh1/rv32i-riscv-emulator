# RV32I RISC-V Emulator

This project is a C++20 emulator for the RV32I RISC-V instruction set.

## Goal

The goal is to build a clean systems programming project that demonstrates:

- CPU state simulation
- memory simulation
- binary instruction decoding
- instruction execution
- testing
- debugging tools
- clean C++ project structure

## Planned Features

- RV32I instruction support
- register file
- program counter
- memory model
- fetch/decode/execute loop
- trace mode
- debugger
- disassembler
- automated tests
- GitHub Actions CI

## Current Status

Project setup in progress.

## Non-Goals for Version 1

The first version will not support:

- RV64
- floating point
- compressed instructions
- Linux booting
- virtual memory
- privileged mode
- interrupts
- multicore execution
