# Project Brief

This project is an RV32I RISC-V emulator in C++20.

The emulator should simulate a simple RISC-V CPU by maintaining registers, memory, and a program counter.

The first goal is to execute a very small instruction such as ADDI.

The long-term goal is to support the RV32I base integer instruction set with tests, trace mode, debugger, and disassembler.
